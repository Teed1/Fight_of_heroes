# Battle Heroes Game (Python)

Небольшая консольная игра, в которой герои сражаются друг против друга.  
Каждый герой имеет здоровье (HP) и атаку. Во время боя урон наносится случайно — от 80% до 120% базового значения атаки.

---

## ⚔️ Функционал

- Создание класса **Hero** (имя, здоровье, урон).
- Класс **Team**, содержащий героев.
- Возможность добавлять героев в команду через оператор `+`.
- Случайный выбор двух героев для битвы.
- Пошаговый бой до смерти одного из героев.
- Вывод хода боя и результатов.
